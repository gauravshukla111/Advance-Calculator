## ✨ Features

- 🎬 Smooth UI Animations (Ripple, Floating, Click Effects)

## ⚙️ How It Works

- User inputs are handled using JavaScript functions.
- Expressions are evaluated dynamically using `eval()`.
- History is stored and displayed in a slide-in sidebar.
- Theme switching is done using class toggling (`dark` / `light`).
- Animations are implemented using CSS transitions & keyframes.

---

## ⚠️ Important Note

> This project uses `eval()` for quick calculations.  
> In production, it's recommended to use a safer library like `math.js`.
